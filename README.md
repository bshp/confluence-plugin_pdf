# PDF Macros for Confluence  
Modified plugin for Baptist School of Health Professions  
  
*Plugin provides 7 different macros to enhance the PDF-export experience:*  
* Add Current DateTime  
* Remove Page Break  
* Set Page Break  
* Set Landscape  
* Set Fontsize  
* Hide in Webview  
* Hide in PDF
